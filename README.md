# Quick Start

## 1. Install Acceleer CLI

Run this command in PowerShell:

```powershell
irm https://github.com/acceleer/acceleer-cli/releases/latest/download/install.ps1 | iex
```

## 2. Clone this repo

```powershell
gh repo clone acceleer/blueprints
```



## 3. Generate PLC Code from FDS

```powershell
cd blueprints
```

```powershell
acceleer generate
```
